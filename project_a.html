Cozy Farm

Cozy Farm is a project aimed at combining relaxation and exploration. This 2D game was developed in Unity and features mechanics such as animals, enemies, tools, NPCs, inventory, farming, crafting system, and time management. From an artistic perspective, it is a 2D game using COZY assets.
