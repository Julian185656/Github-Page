Neighborhood reports

This application was designed for neighborhood reports so that government departments can address them more efficiently and quickly. Different APIs were used, as well as Firebase and Supabase as databases.

Initial Screen

This is a full walkthrough of the application.

Home

The registration and login are displayed. These are validated using a database, along with other details.

Modules

You can see the modules developed within the application.

Reports Module and Viewing Resolved Issues

You can see that reports can be submitted within the application, and it also features an interactive map where you can click on the markers. These represent resolved issues.

Departments and Comments

In this section, you can see the departments to which you can send a message using the WhatsApp API. The comments section shows what neighbors think about the fixes displayed on the map.

View Reports and Edit Credentials

Here you can see the screen to edit previously created reports. On the next screen, you can see the editing of the logged-in user's credentials.
